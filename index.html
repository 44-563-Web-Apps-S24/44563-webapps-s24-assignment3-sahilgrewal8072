The attentive and courteous staff adds to the overall enjoyment, ensuring that each visit is not just a meal but a memorable celebration of Mexican cuisine.

Code Fencing

The below code helps us to Set logging for debug(generally for detecting errors and testing purposes) level along with file name and output to a sample.log file in the respective path. 

import logging

        logger = logging.getLogger(__name__)
        logger.setLevel(logging.DEBUG)
        
        formatter = logging.Formatter('%(asctime)s:%(name)s:%(message)s')
        
        file_handler = logging.FileHandler('sample.log')
        file_handler.setLevel(logging.ERROR)
        file_handler.setFormatter(formatter)
        
        stream_handler = logging.StreamHandler()
        stream_handler.setFormatter(formatter)
        
        logger.addHandler(file_handler)
        logger.addHandler(stream_handler)
